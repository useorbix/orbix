## What is Orbix?

Orbix is a simple, lightweight terminal data lib, used by devs for certain features/cases. If you want free, simple and clean terminal utilities for developers.

The software is made of a variety languages for the user to pick from. Each language, though has different utilities inside it.

## How to use?

Folow the steps to use Orbix:

1. Clone the repo:
   ```bash
   git clone https://github.com/UseOrbix/orbix.git
   ```


2. `cd` into the orbix directory in the orbix git repo:
   ```bash
   cd orbix/orbix
   ```

3. Select your language
4. `cd` into the directory for your chosen language:
   ```bash
   cd [chosen language]
   ```

> [!NOTE]
> Not all laguages exist on Orbix. Use discussions to ask for more.
   

5. Use `ls` to view the files in the language directory
6. Select a terminal application
7. Move the application to the new location:
   ```bash
   mv [chosen application] [new location]
   ```

> [!CAUTION]
> You may have to complete this step more the once as there might be more than one file in each directory.

8. `cd` into the directory of the [new location]:
   ```bash
   cd [directory location]
   ```

9. Run the program

> [!TIP]
> Run an automated script that does these steps for you. Find it [here](utils/automation/automate.sh).
